# 蛋的世界 (Egg World)

一个关于蛋的科普知识网页，展示蛋的多样性、营养价值和有趣事实。

## 项目介绍

这个网页旨在向用户介绍关于蛋的各种知识，包括：

- 蛋的不同种类（鸟类蛋、爬行动物蛋、鱼类蛋、昆虫蛋）
- 蛋的营养价值
- 有趣的蛋知识
- 蛋在烹饪中的应用

## 技术栈

- **HTML5**：页面结构和内容
- **CSS3**：样式设计和动画效果
- **响应式设计**：适配不同屏幕尺寸

## 页面内容

### 1. 英雄区域
- 介绍蛋作为生命起点的重要性
- 包含一个动态浮动的蛋形图案

### 2. 蛋的种类
- 展示不同类型的蛋及其特点
- 鸟类蛋、爬行动物蛋、鱼类蛋、昆虫蛋

### 3. 营养价值
- 详细说明蛋的营养成分
- 包含可视化的营养成分图表

### 4. 有趣的蛋知识
- 分享关于蛋的趣味事实
- 如世界上最大和最小的蛋

### 5. 烹饪中的蛋
- 展示蛋在烹饪中的多种用途
- 煎蛋、煮蛋、炒蛋等常见做法

## 如何使用

1. 直接访问仓库的GitHub Pages链接（如果已启用）
2. 克隆仓库到本地，在浏览器中打开`index.html`文件
3. 修改HTML和CSS文件来自定义内容和样式
